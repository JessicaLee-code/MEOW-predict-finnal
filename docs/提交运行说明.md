# MEOW 提交件运行说明

## 运行环境

- **Python >= 3.8（必需）**
- 基础依赖：`numpy` / `pandas` / `scikit-learn` / `lightgbm` / `tables`（PyTables，读 `.h5`）
- **可选（要拿到融合预测 ~0.09 时需要）：PyTorch 2.6 + NVIDIA GPU（CUDA）**
  - 安装示例：`pip install torch==2.6.0 --index-url https://download.pytorch.org/whl/cu124`
    （`cuXXX` 按本机 `nvidia-smi` 显示的 CUDA 版本选：≥12.4 用 cu124、≥12.6 用 cu126、≥11.8 用 cu118）
  - **未安装 torch 或无 GPU 时**：程序自动回落到纯传统多因子模型（Pearson 约 0.0812），
    **不会崩溃、不会输出 NaN**——传统腿是无条件保底。
  - 如需强制关闭深度学习腿（只跑传统）：设环境变量 `MEOW_DISABLE_DL=1`。

## 运行方式

```bash
cd MEOW/meow
python meow.py
```

- 默认读取数据目录下的 `.h5` 文件（每个文件名即交易日，如 `20230601.h5`）。
- 替换数据文件路径后直接运行即可；**入口会自动适配数据实际所在的时间段**
  （配置日期与数据不相交时按数据范围自动切分 train/eval），无需改入口日期。
- 评测输出三项指标：Pearson Correlation / R² / MSE。

## 预测方案简述

- **传统腿（无条件保底）**：433 维人工特征上 Ridge + LightGBM 等权融合。
- **深度学习腿（有 torch+GPU 时启用）**：截面序列模型直接建模原始盘口 raw 通道，
  与传统腿去相关，等权融合后三项指标一起改善（详见项目报告）。
