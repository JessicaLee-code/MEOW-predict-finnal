1. **结构成立**：corr 全 15 折为正、种子离散极小（std 0.013 vs TCN 0.068），最坏折稳为正（TCN 当年最坏 −0.0384）。**GRU-on-433 确实看见截面 alpha**（433 特征已携带 cross-z/rank）——"喂 433"决策坐实。此为截面模型 ② 的消融下界（③）。
2. **R² 微负 = MSE/corr 矛盾实锤**：纯 MSE(λ=0) 下 corr 正常但 R² 卡在 0 下沿（−0.007），预测量纲未胜过猜均值——低信噪下 MSE 收缩预测的必然，与预期一致；老师 ⅓·R² 当前小幅净负。
3. **过拟合受控**：train_corr ~0.09–0.15 vs val ~0.04–0.08，重正则把 gap 压在中等；best_epoch 4–7，max_epochs=30 从未触顶（下次可降至 ~15 省时）。
4. **vs 传统（仅参考）**：mean 0.0585<0.0776、最坏 0.0391<corr_min 0.0491；同数量级、处处为正，但 λ=0 基线低于传统头条——非定论，正式对决须统一折 + 加 λ·corr。

**下一步启示**：
- λ·corr 是必走杠杆，但 R² 已贴 0 下沿 → 加 corr 会进一步压 R² → **「R²≥0 护栏」是马上咬合的硬约束**，λ 按三指标 Pareto 选（已落 WT-B）。
- **近免费修 R²**：本跑 `postprocess_type=none`；因 corr 对尺度/平移不变 → **训练段拟合线性 rescale/shrinkage（a·ŷ+b，fit-train / apply-val、零泄漏）可把 R² 抬到 ≥0 而 corr 不掉**。已补进 WT-B 简报。
