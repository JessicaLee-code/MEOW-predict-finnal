# 目前严峻形式报告

日期：2026-05-16  
项目：MEOW 金融时序预测 V3.3  
用途：向老师汇报当前实验的真实困难和下一步判断

## 1. 当前结论先说

当前实验形式比较严峻，核心原因不是“没有模型”，而是“正式 rolling 下有效信号强度不足”。

在正式 rolling 口径下，现阶段最好的可复现结果仍然只有大约 `0.048` 左右的相关系数，距离目标 `0.10` 还有明显差距。也就是说：

- 现在不是“再调一点就能到 10%”
- 现在更像是“已经找到一个稳定下限，但上限还没打开”
- 如果继续只在现有 backbone 上做小修小补，预计很难突破到 10%

## 2. 已验证的正式 rolling 结果

正式 rolling 统一设定：

- 5 折 rolling
- `train_window=8`
- `val_window=2`
- `step=10`
- `embargo=1`
- `max_train_days=4`

### 2.1 Backbone 结果

| experiment | rolling_corr_mean | rolling_corr_std | rolling_corr_min | stability_score |
| --- | ---: | ---: | ---: | ---: |
| `B0_ridge_legacy` | 0.036365 | 0.011488 | 0.025569 | 0.028323 |
| `B6_common_residual` | 0.044205 | 0.022870 | 0.016618 | 0.028196 |
| `B7_soft_regime` | 0.041538 | 0.021872 | 0.004875 | 0.026228 |
| `B8_ridge_legacy_plus_core` | 0.037219 | 0.031576 | -0.014326 | 0.015116 |

### 2.2 当前最好结果

- 最稳主 backbone：`B0_ridge_legacy`
- 当前最好 Ridge 增强：`R02_ridge_legacy_plus_norm_core`
  - `rolling_corr_mean = 0.047976`
  - `stability_score = 0.037494`
- 当前最好受限融合：`B0 + B6/B7`
  - `rolling_corr_mean = 0.047179`
  - `stability_score = 0.037353`

### 2.3 现实判断

当前正式 rolling 下的真实水平，大致还是在 `0.036 ~ 0.048` 区间，不在 `0.10` 附近。

## 3. 为什么说形式严峻

### 3.1 backbone 已经验证，但没有打开上限

`B0_ridge_legacy` 是最稳的，但它的均值相关只有 `0.036365`。  
`B6` 和 `B7` 虽然有时单折高，但稳定性并不优于 `B0`。  
`B8` 把 core 直接并进去后，波动反而更大，说明“加特征”并不自动等于“更强信号”。

### 3.2 复杂模块没有形成稳定优势

已经验证过的复杂方向包括：

- `common residual`
- `soft regime`
- `B8 legacy + core`
- `patch_summary`
- `cross_rank_features`

它们的共同问题是：

- 有时单折高
- 但整体 rolling 不稳
- `rolling_corr_std` 偏大
- `rolling_corr_min` 偏低

这意味着它们更像“局部有效”，而不是“可泛化的主结构”。

### 3.3 10% 目标和当前证据不匹配

如果以 `corr=0.10` 作为目标，那么当前证据只能说明：

- 已经存在一些有效信号
- 但信号强度不够
- 现有结构还没有把信号充分释放出来

换句话说，现在离 10% 不是一点点修修补补的问题，而是要重新找更强的信号来源。

## 4. 已经确认的有效信息

虽然整体不够强，但已经确认了一些有价值的事实：

### 4.1 legacy 比 core 更稳

`legacy` 的有效信号主要来自：

- 订单簿 imbalance
- 成交 imbalance
- imbalance 的平滑/组合
- 12 分钟相对强弱偏离

### 4.2 Ridge 系数显示出的主信号

在 `B8` 的系数里，较稳定、较靠前的特征包括：

- `trade_imb`
- `ob_imb4` / `obi4`
- `ob_imb0` / `obi0`
- `order_pressure`
- `trade_imbema5`
- `lagret12`

这说明当前有效信号更偏向“供需压力”和“交易方向压力”，而不是复杂非线性结构。

### 4.3 normalized core 比原始 core 更值得保留

当前最好 Ridge 增强不是 `legacy + core`，而是：

- `legacy + normalized core`

这说明不是 core 完全没用，而是原始 core 的噪声太大，必须做更强约束后才有机会变成稳定增量。

## 5. 当前风险

### 5.1 继续盲目扩复杂模型，风险大于收益

如果继续围绕：

- `soft regime`
- `residual target`
- `postprocess`
- `MLPLOB-lite`
- 更复杂的 stacking / blend

那么更可能得到的是：

- 单折看起来好
- rolling 不稳定
- 最终不可复现

### 5.2 现在最大风险是“过拟合到局部切分”

当前已经出现过这种现象：

- 某些模型在个别 fold 高
- 但整体 stability_score 不如 `B0`

这说明后续不能再用单次 holdout 作为决策依据，否则会继续偏离真实泛化表现。

### 5.3 如果不换信号来源，10% 目标大概率达不到

这不是悲观判断，而是当前数据已经支持的判断：

- backbone 已确认
- expert 已确认只能辅助
- feature enhancement 也只拿到接近 `0.048`

如果不新增更强的信号族，继续在这条线内打磨，突破到 `0.10` 的概率不高。

## 6. 当前阶段建议

建议把接下来的工作分成三层：

### 6.1 先保底

- 以 `B0_ridge_legacy` 作为当前主 backbone
- `B6` / `B7` 仅保留小权重 expert 角色
- 暂停继续扩展复杂 regime / residual / postprocess

### 6.2 再做信号诊断

重点检查：

- 标签构造是否还能进一步优化
- `fret12` 对齐和 embargo 是否足够严格
- 是否存在被当前特征工程压掉的强信号
- 是否有更强的跨截面相对特征没有进入模型

### 6.3 最后才考虑冲 10%

如果未来能找到新的强信号族，再考虑：

- 中等复杂度模型
- 更强但受控的特征组
- 严格 rolling 下的受限融合

## 7. 可直接向老师说明的话

可以直接这样说：

> 目前正式 rolling 下已经确认了稳定 backbone，但稳定上限还只有 0.04 到 0.05 左右，离 0.10 还有明显差距。复杂模块目前主要表现为局部折有效、整体不稳，所以现在更关键的是找更强的信号来源，而不是继续盲目堆模型。

## 8. 结论

当前阶段的真实状态是：

- 有稳定下限
- 没有达到 10%
- 复杂模块还没有形成可复现优势
- 继续冲分必须先找更强信号，而不是继续做小幅模型堆叠
